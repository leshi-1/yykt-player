# YYKT Auto Player

蕴瑜课堂视频自动播放 Chrome 扩展。

## 功能

- **自动播放** — 进入视频页面自动静音播放
- **验证自动按住** — 检测"按住验证"按钮，自动模拟长按通过
- **进度感知切换** — 读取"我的播放进度"，达到 90% 自动跳下一集
- **断点记忆** — 记住上次播放的视频，打开首页自动跳回
- **防挂起** — 自动恢复浏览器暂停的视频，防止休眠中断
- **状态栏** — 页面顶部显示播放状态和时间

## 安装

1. 下载本仓库 ZIP 或 `git clone`
2. Chrome 打开 `chrome://extensions/`
3. 右上角开启 **"开发者模式"**
4. 点 **"加载已解压的扩展程序"**
5. 选择 `yykt-extension` 文件夹
6. 打开 https://courses.gdut.edu.cn/ 即可

## 使用

安装后打开蕴瑜课堂视频页面，脚本自动生效：

- 顶部状态栏显示当前状态和时间
- 视频自动静音播放
- 出现"按住验证"时自动处理
- 看完一集自动跳下一集

刷新页面（F5）可强制回到记住的视频。

## 项目结构

```
yykt-extension/
├── manifest.json    # 扩展配置
└── content.js       # 核心脚本
```

## 技术细节

- Manifest V3
- 适配 courses.gdut.edu.cn 页面结构
- 含 iframe 内视频检测
- 使用 MediaSession API 保持播放活跃
- MutationObserver 监听页面动态变化
